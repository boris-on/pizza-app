1. Корзина: Пользователи должны иметь возможность добавлять несколько товаров в корзину и удалять товары из нее.
2. Геолокация: Приложение должно автоматически определять местоположение пользователя.
3. Методы оплаты: Приложение должно поддерживать различные методы оплаты, включая оплату при заказе и оплату наличными.
4. Комментарии к заказу: Пользователи должны иметь возможность добавлять комментарии к заказу.
5. Ночная тема: Приложение должно иметь возможность переключения на ночную тему.
6. Минимальная авторизация и установка аватарки: Пользователи должны иметь возможность авторизоваться с минимальными данными и устанавливать аватарку.
7. Рекламный блок от спонсора: Приложение должно иметь рекламный блок от спонсора.
8. Определение зон доставки: После выбора на карте, приложение должно определить, куда можно осуществлять доставку, а куда нельзя.
9. Фильтрация пицц по ингредиентам: Пользователи должны иметь возможность фильтровать пиццы по ингредиентам.
10. Фильтрация пицц по калориям: Пользователи должны иметь возможность фильтровать пиццы по калорийности.
11. Сортировка по цене и названию: Пользователи должны иметь возможность сортировать пиццы в каталоге по цене и названию.
12. Три размера пиццы: Пиццы должны быть доступны в трех размерах.
13. Карточка пиццы: Каждая карточка пиццы должна содержать фотографию, описание и продающий текст.
14. Управление количеством в корзине: Пользователи должны иметь возможность управлять количеством товаров в корзине.
15. Быстрый заказ без авторизации: Пользователи должны иметь возможность быстро оформить заказ без необходимости авторизации.
16. Отправка чека на почту: После оформления заказа, пользователю должен быть отправлен чек на указанную почту.
17. Подсчет суммы при добавлении товара в корзину: При добавлении товара в корзину, приложение должно автоматически подсчитывать сумму заказа.
